## Definition of done

Scrumin määrittelyn mukainen definition of done sisältää:

- user storyn määrittelyn,
- suunnittelun,
- toteutuksen,
- automatisoitujen testien tekemisen,
- integroinnin muuhun sovellukseen,
- dokumentoinnin.
